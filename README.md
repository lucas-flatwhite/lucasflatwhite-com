# lucasflatwhite.com

Astro-based portfolio site for lucasflatwhite, with homepage content driven by structured data and project markdown entries.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run check
npm test -- --run
```

## Content Editing

- Update homepage profile text, primary links, and recommended commands in `src/data/site.ts`.
- Edit project entries in `src/content/projects/*.md`.
- Adjust section structure and presentation in `src/components/*`.
- Refine shared visual styling in `src/styles/global.css`.
